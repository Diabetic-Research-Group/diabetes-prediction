# Tests Directory

This directory contains unit tests and integration tests for the project.

## Structure

Mirror the structure of the `src/` directory:
- `test_data.py` - Tests for data processing functions
- `test_features.py` - Tests for feature engineering
- `test_models.py` - Tests for model training and prediction

## Running Tests

Run all tests with:
```bash
pytest tests/
```

Run specific test file:
```bash
pytest tests/test_data.py
```
