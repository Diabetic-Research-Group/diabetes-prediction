# Models Directory

This directory stores trained model files and serialized model objects.

## Storage Format

Common formats for saving models:
- `.pkl` - Pickle files for scikit-learn models
- `.h5` or `.keras` - Keras/TensorFlow models
- `.pt` or `.pth` - PyTorch models
- `.joblib` - Joblib serialization for large numpy arrays

## Naming Convention

Use descriptive names that include model type and date:
- `diabetes_random_forest_2024_01_15.pkl`
- `diabetes_neural_net_v1.h5`
