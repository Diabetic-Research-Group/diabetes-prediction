# Notebooks Directory

This directory contains Jupyter notebooks for exploratory data analysis, experimentation, and visualization.

## Naming Convention

Use the following naming convention for notebooks:
- `01_data_exploration.ipynb` - Initial data exploration
- `02_feature_engineering.ipynb` - Feature creation and selection
- `03_model_training.ipynb` - Model training experiments
- `04_model_evaluation.ipynb` - Model performance evaluation

Number prefixes help maintain logical order of analysis workflow.
